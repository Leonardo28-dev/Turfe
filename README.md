🏇 Turfe Amigo
📖 Sobre o Projeto

O Turfe Amigo é uma aplicação web desenvolvida para simular corridas de cavalos de forma dinâmica e interativa. A cada volta, os cavalos recebem tempos aleatórios entre 7,0 e 9,0 segundos, que são acumulados até o final da corrida.

O vencedor é o cavalo que concluir o percurso com o menor tempo total.

✨ Funcionalidades
🐎 Cadastro de 2 a 6 cavalos
📝 Definição do nome de cada cavalo
🔄 Escolha da quantidade de voltas da corrida
🎲 Sorteio automático dos tempos em cada volta
📊 Exibição dos tempos parciais e acumulados
🏆 Pódio com os 3 melhores colocados
